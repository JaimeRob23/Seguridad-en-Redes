## Solucion
- Solucione este problema usando el comando curl, el cual es un get, el truco aqui es despues de haber inspeccionado que elementos contiene la cookie, encontramos el parametro de admin, al invocar el sitio con curl le enviamos una cookie modificada con el admin True, haciendo que la pagina muestre la flag:
  picoCTF{th3_c0nsp1r4cy_l1v3s_4d184b0d}
