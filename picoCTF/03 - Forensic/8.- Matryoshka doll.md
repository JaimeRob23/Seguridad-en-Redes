## Matryoshka doll
Matryoshka dolls are a set of wooden dolls of decreasing size placed one inside another. What's the final one? Image: [dolls.jpg](https://challenge-files.picoctf.net/c_wily_courier/b236ead0cb75e5a4b8b8d5aa15c6cc80369b3ea778f1a116911f23d774a92bf4/dolls.jpg)

## Solucion
- Al leer los meta datos de la imagen nos damos cuenta que tiene informacion binaria guardada, usamos el comando de binwalks para ver que guarda en su interior:
```
┌──(lighttwo33㉿HackJR)-[~/Descargas]
└─$ binwalk dolls.jpg            

DECIMAL       HEXADECIMAL     DESCRIPTION
--------------------------------------------------------------------------------
0             0x0             PNG image, 594 x 1104, 8-bit/color RGBA, non-interlaced
3226          0xC9A           TIFF image data, big-endian, offset of first image directory: 8
272492        0x4286C         Zip archive data, at least v2.0 to extract, compressed size: 378933, uncompressed size: 383920, name: base_images/2_c.jpg
651591        0x9F147         End of Zip archive, footer length: 22


```
- Descubrimos que tiene zip guardado dentro de la imagen, al hacerle unzip nos saca otra imagen, le volvemos a hacer unzip varias veces hasta que la ultima magen nos arroja la flag:
```
┌──(lighttwo33㉿HackJR)-[~/Descargas/base_images/base_images/base_images]
└─$ ls            
4_c.jpg  flag.txt
                                                                             
┌──(lighttwo33㉿HackJR)-[~/Descargas/base_images/base_images/base_images]
└─$ cat flag.txt    
picoCTF{LL9lb1dR4QbGe4l4iWCvGq9pdtwt7392}

```
