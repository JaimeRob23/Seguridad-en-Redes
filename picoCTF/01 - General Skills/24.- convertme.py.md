## convertme.py
Run the Python script and convert the given number from decimal to binary to get the flag. [Download Python script](https://artifacts.picoctf.net/c/24/convertme.py)

## Solucion 
- Al correr el programa te pide convertir un decimal a una base binaria, en mi caso fue 43 y lo unico que hice fue hacer la convercion a base binaria (ir dividiendolo entre 2 y sacando su modulo)
```

└─$ python convertme.py 
If 43 is in decimal base, what is it in binary base?
Answer: 101011
That is correct! Here's your flag: picoCTF{4ll_y0ur_b4535_722f6b39}

```
